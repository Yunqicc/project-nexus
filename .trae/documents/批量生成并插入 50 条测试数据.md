## 目标
- 向 Supabase 的 `guestbook` 表批量插入 50 条测试数据
- 数据需符合表约束：nickname (1-20字符), content (1-50字符)

## 实现方案
1. **安装依赖**：`@faker-js/faker` 用于生成真实的模拟数据
2. **编写脚本**：创建 `scripts/seed.ts`
   - 读取 `.env.local` 中的 Supabase URL 和 Key
   - 使用 `faker` 生成 50 条随机留言（中文/英文混合）
   - 调用 `supabase.from('guestbook').insert([...])` 批量写入
3. **执行脚本**：通过 `ts-node` 或 `vite-node` 运行该脚本

## 脚本逻辑细节
- 昵称：随机生成用户名，截断至 20 字符以内
- 内容：随机生成短句，截断至 50 字符以内
- 错误处理：捕获并打印插入过程中的错误

- 执行完成后，你只需刷新前端页面，即可看到滚动加载或列表展示的效果
